# Climate Adaptation and Security in the Sahel and Horn of Africa
## Global Center on Adaptation and Bodhi Global Analysis


1. 'data_preprocessing.py' shows how the data have been processed. Note that some dataset paths were changed during preprocessing, so running the code all at once may cause errors. If you want to run the code, please update the dataset paths accordingly before executing
2. Some raw datasets are not included in this repository due to size limitations: UCDP_GED and ODA_Total_Net_from_Official_Donors
3. Following datasets have been used for this study:
 - UCDP Georeferenced Event Dataset (UCDP GED) + UCDP datasets
 - UN OCHA - Humanitarian Needs
 - UN OCHA - Subnational Administrative Boundaries
 - SIPRI - Military Expenditure Database
 - WORLD Policy Analysis Center (WORLD) -  Climate Adaptation Data
 - EM-DAT -  Emergency Events Database
 - IPC - Acute Food Insecurity Country Data
 - OCHA - Global Subnational Population Statistics
 - Ibrahim Index of African Governance (IIAG)
 - Global Data Lab
 - WFP - Rainfall Indicators at Subnational Level
 - OPHI - Global MPI
 - World Bank - Agriculture and Rural Development
 - World Bank - Climate Change Indicators
 - ND-GAIN Country Index (Notre Dame Global Adaptation Initiative)
 - INFORM Risk Index (EU JRC / OCHA)
 - OECD - Official Development Assistance (ODA)
4. 'statistical_analysis.ipynb' demonstrates the process of the statistical analyses conducted
