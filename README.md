# Climate Adaptation and Security in the Sahel and Horn of Africa
## Global Center on Adaptation and Bodhi Global Analysis


1. 'data_preprocessing.py' shows how the data have been processed. Note that some dataset paths were changed during preprocessing, so running the code all at once may cause errors. If you want to run the code, please update the dataset paths accordingly before executing

3. 'statistical_analysis.ipynb' demonstrates the process of the statistical analyses conducted
